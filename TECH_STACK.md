# 技術スタック — QND Lab Website Redesign

千葉大学 量子ナノデバイス研究グループ（青木研究室）ホームページ再開発プロジェクトの技術構成。

---

## スタック一覧

| レイヤー | 技術 | 選定理由 |
|---|---|---|
| 静的サイトジェネレータ | **Hugo** | ビルド速度が極めて高速（数百ページを数秒で生成）。Go製シングルバイナリで環境構築が容易。多言語（i18n）ネイティブサポート。 |
| テンプレートエンジン | Go Templates（Hugo内蔵） | Hugo標準。追加依存なし。 |
| CSSフレームワーク | **Tailwind CSS** | ユーティリティファーストで一貫したデザインを維持しやすい。PurgeCSSによる未使用CSS自動削除でファイルサイズ最小化。 |
| コンテンツ管理 | **Markdown + YAML Front Matter** | 研究室メンバーがテキストエディタで編集可能。Gitでの変更管理と相性が良い。 |
| データファイル | **YAML / JSON** | メンバー情報・論文リスト・設備データなどの構造化データを管理。Hugo Data Templatesで自動レンダリング。 |
| バージョン管理 | **Git + GitHub** | コンテンツの変更履歴管理。プルリクエストによるレビューフロー。 |
| ホスティング | **GitHub Pages / Cloudflare Pages** | 無料で高可用性。CDN配信による高速表示。SSL/TLS自動対応。 |
| CI/CD | **GitHub Actions** | mainブランチへのpush時に自動ビルド・デプロイ。PR毎のプレビュー環境自動生成。 |
| 画像最適化 | **Hugo Pipes + WebP変換** | 画像のリサイズ・WebP変換をビルド時に自動実行。 |
| 静的検索 | **Pagefind** | ビルド時にインデックスを生成するクライアントサイド検索。外部サービス不要。日本語対応。 |

## Hugo 選定理由（比較）

動的CMS（WordPress等）ではなく静的サイトジェネレータを採用した理由：

- **セキュリティ：** 静的ファイル配信のため、SQLインジェクション・XSS等の攻撃ベクトルが存在しない。サーバー管理コストゼロ。
- **コスト：** GitHub Pages / Cloudflare Pages によりホスティング費用が無料。
- **パフォーマンス：** CDN経由の静的配信により、世界中どこからでも高速アクセス可能。
- **保守性：** Node.js・PHPのランタイム更新、DBパッチ適用といった運用コストが発生しない。

| 比較対象 | 言語ランタイム | 多言語サポート | ビルド速度 |
|---|---|---|---|
| **Hugo** | Go（シングルバイナリ） | ネイティブ対応 | 最速 |
| Jekyll | Ruby | プラグイン依存 | 遅い（大規模時） |
| Next.js | Node.js | 手動実装が多い | 中程度 |

## SEO対策

- `title`、`meta description`、OGPタグ（`og:title`, `og:description`, `og:image`）を全ページ自動生成
- JSON-LD形式の構造化データ（`Organization`, `ResearchProject`, `Person`）を埋め込み
- 論文ページに Google Scholar 向けメタタグ（`citation_title`, `citation_author`, `citation_publication_date` 等）を付与
- `sitemap.xml` と `robots.txt` を自動生成
- `hreflang` 属性による日英ページの言語関連付け
- `canonical` URL を全ページに設定し、重複インデックスを防止

---

## パフォーマンス対策

- 画像の WebP 変換・レスポンシブ画像（`srcset`）の自動生成
- PurgeCSS による未使用スタイル削除、HTML/CSS/JS の minify
- フォント `font-display: swap` 指定による FOIT 防止、サブセット化
- `loading="lazy"` を全画像に適用
- HTTP/2 + Brotli 圧縮（Cloudflare Pages 使用時は自動適用）

**目標値：**
- Lighthouse Performance Score: **95以上**
- First Contentful Paint: **1.0秒以下**
